SkyMixer

Multi-track MIDI mixer & FX processor
input MIDI file

SkyMixer の使い方

基本操作

  • MIDIファイルをドラッグ＆ドロップまたはクリックで読み込み
  • 複数ファイル対応 — 複数のMIDIファイルを同時に読み込み、トラックをマージできます
  • 最初のファイルのテンポ・拍子・キーがマスタータイムラインとして使用されます
  • ▶ 再生 / 一時停止、⏹ 停止、🔁 ループ再生
  • プログレスバーをクリックで任意の位置にシーク

トランスポート

  • BPM — テンポを手動変更可能（元のテンポからオーバーライド）
  • KEY — グローバルトランスポーズ（半音単位、±12）。ドラムトラックには適用されません
  • レンジ選択 — プログレスバーの START / END マーカーで再生範囲を指定
  • ループON時にレンジ範囲をリピート再生
  • Bar:Beat 表示でリアルタイムに現在位置を確認

トラック操作

  • M ミュート — トラックを無音に
  • S ソロ — そのトラックだけを再生
  • L ── R パンスライダー — 左右の定位を調整（リアルタイム反映）
  • ボリュームスライダー — 各トラックの音量を調整（0〜200%）
  • レベルメーター — 各トラックの出力レベルをリアルタイム表示
  • トラック名をクリックするとピアノロール表示を開閉

楽器・オクターブ・和音フィルター

  • インストゥルメント変更 — GM 128楽器から選択。ドラムトラックも変更可能です
  • 🥁 Standard Kit (Drums) — ドラムキットとして再生（オクターブ/トランスポーズ無効）
  • oct ＋/− — トラック単位のオクターブシフト（±3、ドラム時は無効）
  • 和音フィルター（♪ All / ↑ Top / ↓ Bass）：
    • ♪ All — すべてのノートをそのまま再生
    • ↑ Top — 最高音のメロディラインだけを抽出して再生
    • ↓ Bass — 最低音のベースラインだけを抽出して再生
  • 和音フィルターはWAV/MIDIエクスポートにも反映されます

FXチェーン

IN → NOTE → [ドラッグ順序変更可能なFX] → PEQ → OUT

  • NOTE（固定・先頭） — アタック・ベロシティ・デュレーション・リリースを調整
  • EQ — 7バンドグラフィックイコライザー（60Hz〜12kHz、±12dB）
  • CHORUS — ステレオデュアルボイスコーラス（Rate / Depth / Spread / Mix）
  • DYN — ダイナミクスコンプレッサー（Threshold / Ratio / Attack / Release / Makeup）
  • ROOM — ディレイ＆リバーブ（Time / Feedback / Size / Mix）
  • PEQ（固定・末尾） — 5バンドパラメトリックイコライザー（ビジュアル操作）

中央の4つのFX（EQ / CHORUS / DYN / ROOM）はドラッグで順序を変更できます

パラメトリックEQ の操作

  • コントロールポイントを左右にドラッグ → 周波数を変更
  • コントロールポイントを上下にドラッグ → ゲインを変更（±18dB）
  • 下部の回転ノブを上下にドラッグ → Q値を変更（0.1〜10.0）
  • バンド構成: ① LS（Low Shelf）② LM ③ MID ④ HM ⑤ HS（High Shelf）
  • 再生中はリアルタイムのスペクトラムアナライザーが背景に表示されます

レベルメーター

  • 画面下部に10バンド ステレオ レベルメーターを搭載
  • L/R独立でリアルタイムのオーディオレベルを表示
  • 各トラックにも個別のレベルメーターを表示
  • マスター出力スライダーで全体音量を調整

エクスポート

  • ⬇ Export MIDI — ミュート/ソロ/ボリューム/パン/オクターブ/トランスポーズ/楽器変更を反映
  • ミュートトラック除外オプション付き（トラック構造維持 or 圧縮書き出し）
  • Export WAV — すべてのFXエフェクト・パン設定を反映したオーディオをレンダリング
  • WAVエクスポートにはNOTE・EQ・CHORUS・DYN・ROOM・PEQ の全設定と和音フィルターが適用されます

💡 ヒント

  • 全トラックのピアノロールを開くと各パートのノート配置を一覧できます
  • NOTEエフェクターのDurationとReleaseを上げると、短く切れる音を自然に伸ばせます
  • PEQでリアルタイムにスペクトラムを見ながら帯域を調整し、そのままWAVエクスポートできます
  • 各FXにはON/OFFトグルとOUTPUTレベルがあり、個別にバイパス可能です
  • 和音フィルター「↑ Top」で伴奏付きトラックから主旋律だけを取り出せます
  • 複数のMIDIファイルをマージして、異なるファイルのトラックを同時にミキシングできます
